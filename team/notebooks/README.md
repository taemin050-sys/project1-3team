# team/notebooks/ — 승격된 서사 노트북 (제출물)

마일스톤 심사를 통과해 **승격된 노트북만** 올라온다. 이 노트북 자체가 **팀 공통 채점 대상 제출물**이다.

## 규약
- **작업용 노트북은 `beamsearch/<이니셜>/`.** 여기엔 승격 선정된 것만.
- 승격 시 **의사결정·토론을 마크다운 셀로 인레이** — 결과와 그 결과에 이른 사고 과정을 한 흐름으로.
  (누가 무엇을 왜 선택/기각했는지 → 협업 과정·과정 우수성 증거, docs/03 ④)
- 코드 셀 + 출력 + 중간 리포트 + 의사결정 md = 완결된 서사.
