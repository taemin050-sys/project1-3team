## S6b. 단계별 합성 파이프라인 고도화 (기본크롭 → Grabcut → SAM2 하이브리드)
- **1단계 기본크롭컷**(사각 이음새) → **2단계 Grabcut**(외곽 따내기, 반투명 실패) → **3단계 SAM2 하이브리드**(CV 실패분만 SAM 라우팅 + WB정규화 + 정품배경 + 알파컴포지트)
- **3단계 이미지처리 — 포토샵 직관을 CV로 등가 구현**:

  | 포토샵 아이디어 | CV 등가 | 고치는 결함 | 비용 |
  | --- | --- | --- | --- |
  | feather/경계블러 | 경계 alpha 블렌딩 | A만(내부평면 잔존) | 저 |
  | 힐링툴(content-aware) | **cv2.inpaint (Telea/NS)** | A+B 동시(텍스처 전파) | 저 |
  | 스탬프툴 | 배경패치 복사+페더 | A+B(진짜 grain 보존) | 중 |
  | 배경 분산 기반 생성 | median + 지역 노이즈 std 매칭 | A(페더)+B | 중 |

- **단계별 예시**: [이미지: report_12_aug_stage1_basic_crop.png] 1단계 기본크롭 초기실패 · [이미지: report_13_aug_stage2_grabcut.png] 2단계 Grabcut
- **3단계 세부**: [이미지: report_14_aug_stage3_sam_cutout_audit.png] SAM 컷아웃 감사 · [이미지: report_15_aug_stage3_sam_cutout_assets.png] 컷아웃 자산 · [이미지: report_16_aug_stage3_bg_patches.png] 배경패치 · [이미지: report_17_aug_stage3_bg_generated.png] 배경생성 · [이미지: report_18_aug_stage3_composite.png] 최종 합성(유효 synth696, +0.18 주역)

## S7. 평가 신뢰도 — 3대 함정 (핵심 스토리)

## S8b. 성능 레버 판정 — 중간 스냅샷 → 최종 재정합 (56→118)
- **레버 판정(base 시점 → 최종)**:

  | 축 | 중간 판정 | 최종 업데이트 |
  | --- | --- | --- |
  | 용량 11n→11s | ✅ 유일 레버 | **11m 역전** — 데이터 확장으로 11m 최고 단일 0.9988 |
  | 해상도 640→960 | ❌ | 불변(폐기) |
  | 데이터 물량·분포 | ❌ | 물량 무효, 단 커버리지(56→118)가 최대 레버 |
  | 배경다양성 bg64 | ❌ 기각 | 불변 |
  | real copy-paste | ⏳ | 누수 확정 → 무효화 |

- **핵심**: '효과 없다'가 아니라 **'그 데이터 규모에선 아직'**이던 축(용량)과, 규모 불문 불변인 축(해상도)을 구분 — 11m '보류' 판정이 데이터 확장으로 예측대로 실현
- **튜닝 항목별 판정**(비용 대비): 프리징 낮음(8,764장→풀 파인튜닝) · LR/하이퍼 낮음(디폴트 강함) · **에폭 수렴 높음**(11m@100) · box loss 중간(mAP@0.75 특화) · Optuna 스킵(비용≫이득) · **오류분석 최고** · Grad-CAM은 XAI용
- 화자노트: 점수 레버 아닌 곳에 시간 안 씀(시간낭비 함정 회피).
