# beamsearch/ — 탐색 (각자 세계)

빔서치 협업 루프의 **탐색(EXPLORE)** 공간. 팀원별 개인 폴더에서 **구조·도구·방식 모두 자유**롭게
데이터·전처리·모델링·학습을 병렬 실험한다. 작업용 노트북도 여기.

```
beamsearch/
├── LTM/  (이태민)   ├── LHK/  (이형기·KAI)   ├── CJY/  (최중열)
├── JHB/  (장한빈)   └── HWS/  (홍우석)
```

## 규약 (딱 3가지만 공유)
1. **채점기·가드·SSOT는 `shared`를 재사용** — 성능 비교의 공정성 확보. (`from shared.eval.local_map ...`, `from shared.guards.banned import is_banned`)
2. **금지 데이터 가드는 필수** — 어떤 실험이든 `shared.guards`를 통과시킨다.
3. **그 외는 전부 자유** — 폴더 내부 구조·라이브러리·증강·모델·코드 스타일 제약 없음.

## 승격
마일스톤마다 결과·과정을 전원이 비교·심사 → 우수안만 `team/`으로 승격.
승격 대상이 된 노트북은 의사결정 마크다운을 정리해 `team/notebooks/`로 올린다.

> 대용량 산출물(가중치·예측 raw)은 개인 폴더에 쌓지 말 것 → `team/outputs/`(미커밋) 또는 외부 스토리지.
